How to Use This Page
Fallback Image Click the thumbnail image in the preview column to open the full-size fallback image in a new browser window. You can download the image directly from there.
Launch Banner Click the Launch Banner link in the Banner column to open and preview the animated banner at full size in an overlay. Use the Replay button to restart the animation and the Close button to dismiss.
Show Files Click the Show Files button in the Files column to expand a list of all source files included in that banner unit. Click Hide Files to collapse the list.
Download Zip Click the Download Zip button to download a compressed archive of all source files for that banner size directly to your computer.
Design Deck Click the Design Deck button at the top of the page to open the project's creative brief and design reference document in a new browser window.
Validate Click the Validate button to open the HTML5 validation results for this project in a new browser window. This confirms the banner code meets current web standards.
Banner Navigation Use the navigation bar at the top of the page to jump directly to any banner size section. The active section is highlighted as you scroll.
Project Report Click the Project Report link in the top bar to open a full production summary for this project in a new browser window. The report includes banner counts, file totals, status overview, timeline, and a full banner breakdown.
Clickthrough URL Click Show Files on any banner row to expand the file drawer. The Clickthrough URL link on the right side of the drawer opens the banner's destination URL in a new browser window.
